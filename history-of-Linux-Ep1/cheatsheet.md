# Linux Fundamentals – Episode 01 Cheat Sheet
## History & Philosophy of Linux

---

## 🕰️ UNIX → Linux Timeline

### UNIX
- Developed in **1960s** at **Bell Labs**
- Multi-user, multitasking OS
- Written in **C** for portability
- Key people:
  - Dennis Ritchie (C language)
  - Ken Thompson (UNIX)

---

### Linux
- Created in **1991** by **Linus Torvalds**
- Started as a **kernel**
- Built as a **free alternative** to UNIX
- Inspired by UNIX but **not UNIX**

---

## 🧩 GNU & Open Source
- GNU = “GNU’s Not Unix”
- Goal: Free software for everyone
- Linux kernel + GNU tools = Linux OS
- Licensed under **GPL (General Public License)**

---

## 🧠 Core Linux Philosophy

| Principle | Meaning |
|--------|--------|
| Open Source | Code is public & modifiable |
| Everything is a File | Devices, processes, configs |
| Hierarchical FS | Root starts at `/` |
| Multi-user | Many users at once |
| Multitasking | Many processes at once |
| Community Driven | Built by global contributors |

---

## 📁 Linux File System Basics
- Root directory: `/`
- Common directories:
  - `/bin` – essential binaries
  - `/etc` – configuration files
  - `/home` – user home directories
  - `/var` – logs & variable data
  - `/usr` – user programs
