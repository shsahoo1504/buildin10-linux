## 🕰️ Historical Background

### UNIX (1960s)
- Developed at **Bell Labs**
- Designed as a **multi-user, multitasking** operating system
- Written in **C**, making it portable
- Influenced many commercial operating systems

Key contributors:
- **Dennis Ritchie** – Creator of C
- **Ken Thompson** – UNIX co-creator

---

### Rise of Linux (1991)
- Created by **Linus Torvalds**, a student in Finland
- Built as a **free alternative** to expensive UNIX systems
- Initially started as a **kernel**, not a full OS
- Inspired by UNIX but **not UNIX**

---

### GNU & Open Source
- GNU Project aimed to create **free software**
- Linux kernel + GNU tools = complete OS
- Licensed under **GPL (General Public License)**
- Encourages sharing, modification, and collaboration

---

## 🧠 Philosophy of Linux

- **Open Source** – Source code is freely available
- **Everything is a File** – Devices, processes, configs
- **Hierarchical File System** – Root starts at `/`
- **Multi-user** – Multiple users can work simultaneously
- **Multitasking** – Multiple processes run at the same time
- **Community-driven development**

## 🧩 Common Linux Terminologies

| Term | Description |
|-----|------------|
| Kernel | Core of the operating system |
| Distribution | Linux OS with tools and packages |
| Bootloader | Program that starts the OS |
| Service | Background running process |
| File System | Organizes and stores data |
| Shell | Command-line interpreter |
| Desktop Environment | GUI layer (GNOME, KDE, XFCE) |
