### UNIX (1960s)
- Developed at **Bell Labs**
- Designed as a **multi-user, multitasking** operating system
- Written in **C**, making it portable
- Influenced many commercial operating systems

Key contributors:
- **Dennis Ritchie** â€“ Creator of C
- **Ken Thompson** â€“ UNIX co-creator

---

### Rise of Linux (1991)
- Created by **Linus Torvalds**, a student in Finland
- Built as a **free alternative** to expensive UNIX systems
- Initially started as a **kernel**, not a full OS
- Inspired by UNIX but **not UNIX**

---

### GNU & Open Source
- GNU Project aimed to create **free software**
- Linux kernel + GNU tools = complete OS
- Licensed under **GPL (General Public License)**
- Encourages sharing, modification, and collaboration
